# BOOKING APP - COMPLETE BUILD SPECIFICATION

## Objective

Build a simple booking application using:

* Next.js 15 App Router
* TypeScript
* TailwindCSS
* PostgreSQL
* Prisma ORM
* React Hook Form
* Zod Validation
* Google Maps Places API
* Google Maps Distance Matrix API

The application will be deployed to Vercel.

This is a technical assessment project intended to demonstrate:

* Frontend development
* Backend API development
* Database integration
* Form validation
* Google Maps integration
* Clean architecture
* Responsive UI

Do not over-engineer the solution.

Focus on clean code and maintainability.
